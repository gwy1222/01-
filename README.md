## 01. 操作系统概述
在硬件（计算机）和软件（程序）中间的就是操作系统  
software（.c -> .s -> .o -> a.out） -- OS -- hardware（(电路上有)计算机（指令序列）->电路）  
操作系统 = 对象 + API
